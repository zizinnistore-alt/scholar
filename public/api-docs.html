NEXUS API V1

Integrate our AI-driven academic analysis engine directly into your applications.

POST /api/analyze

Analyzes an academic author using Semantic Scholar data and Gemini AI to generate a relevance report.

Request Body (JSON) 
{
  "authorId": "123456789",
  "userDescription": "Looking for experts in quantum cryptography."
}
Response 
{
  "author": { "name": "John Doe", "hIndex": 45, ... },
  "analysis": {
    "match_score": 85,
    "full_report": "...",
    "key_technologies":
  }
}
GET /api/search?q={query}

Searches the global database for researchers matching the query string.

Example 
GET /api/search?q=machine+learning
